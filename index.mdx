## Developer Quickstart

Get your documentation site up and running in minutes

```typescript
import QuiverAI from "quiverai";
const client = new QuiverAI();

const logo = await client.svgs..generations.create({
  model: "arrow-0.5",
  input: "A logo for the next AI Design startup",
});
```
