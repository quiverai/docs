After you complete this setup, you are ready to create your first SVGs using the QuiverAI API.

## Before you begin

You first need a QuiverAI account. To create your account go to [quiver.ai/start ](https://quiver.ai/start)and complete the onboarding. The QuiverAI platform offers a Playground to explore our models and manage your API keys, usage and other configurations. To learn more about the QuiverAI platform visit the [Platform Docs](/platform/overview).

      <Tab title="REST API">
        Currently we only provide official SDK support for Node.js. To call the QuiverAI API with any other language directly use HTTP requests.

        <CodeGroup>

        ```bash cURL
        curl --request POST \
          --url https://api.quiver.ai/v1/svgs/generations \
          --header 'Authorization: Bearer <token>' \
          --header 'Content-Type: application/json' \
          --data '
        {
          "n": 1,
          "top_p": 1,
          "stream": false,
          "temperature": 1,
          "frequency_penalty": 0,
          "presence_penalty": 0,
          "aspect_ratio": "1:1"
        }
        '
        ```

        ```python Python
        import requests
        
        url = "https://api.quiver.ai/v1/svgs/generations"
        
        payload = {
            "n": 1,
            "top_p": 1,
            "stream": False,
            "temperature": 1,
            "frequency_penalty": 0,
            "presence_penalty": 0,
            "aspect_ratio": "1:1"
        }
        headers = {
            "Authorization": "Bearer <token>",
            "Content-Type": "application/json"
        }
        
        response = requests.post(url, json=payload, headers=headers)
        
        print(response.text)
        ```

        
        ```php PHP
        <?php
        ```

        ```go Go
        package main
        
        import (
        	"fmt"
        	"strings"
        	"net/http"
        	"io"
        )
        
        func main() {
        
        	url := "https://api.quiver.ai/v1/svgs/generations"
        
        	payload := strings.NewReader("{\n  \"n\": 1,\n  \"top_p\": 1,\n  \"stream\": false,\n  \"temperature\": 1,\n  \"frequency_penalty\": 0,\n  \"presence_penalty\": 0,\n  \"aspect_ratio\": \"1:1\"\n}")
        
        	req, _ := http.NewRequest("POST", url, payload)
        
        	req.Header.Add("Authorization", "Bearer <token>")
        	req.Header.Add("Content-Type", "application/json")
        
        	res, _ := http.DefaultClient.Do(req)
        
        	defer res.Body.Close()
        	body, _ := io.ReadAll(res.Body)
        
        	fmt.Println(string(body))
        
        }
        ```
