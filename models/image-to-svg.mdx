## Making a request

Use image vectorization to convert a raster image (PNG/JPEG/WebP) into an SVG.

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://api.quiver.ai/v1/svgs/vectorizations \
  --header 'Authorization: Bearer <QUIVERAI_API_KEY>' \
  --header 'Content-Type: application/json' \
  --data '
{
  "model": "arrow-0.5",
  "auto_crop": true,
  "image": {
    "url": "https://example.com/logo.png"
  }
}
'
```


```typescript Node.js (fetch)
const res = await fetch("https://api.quiver.ai/v1/svgs/vectorizations", {
  method: "POST",
  headers: {
    Authorization: "Bearer <QUIVERAI_API_KEY>",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    model: "arrow-0.5",
    auto_crop: true,
    image: { url: "https://example.com/logo.png" },
  }),
});

if (!res.ok) {
  throw new Error(await res.text());
}

const data = await res.json();
console.log(data.data?.[0]?.svg);
```

</CodeGroup>
